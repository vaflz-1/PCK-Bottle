# 🍶 PCK Bottle

**Нативный macOS‑редактор пакетов Godot `.pck` — просмотр, правка, переупаковка и установка модов без терминала.**

PCK Bottle открывает игру на Godot (бандл `.app` или отдельный `.pck`),
показывает содержимое деревом как в Finder и позволяет перетаскивать файлы для
добавления/замены, заранее видеть, что изменится, делать резервную копию и
переупаковывать — а также вытаскивать файлы из `.pck` обратно на диск.

> Это не просто установщик русификаторов, а универсальный менеджер модов для
> PCK в стиле Finder — поддерживает пакеты **Godot 3 и Godot 4** и любой контент
> (текстуры, сцены, скрипты, данные `.import`, локали, …).

## ⬇️ Скачать и установить

1. Возьми свежий **`PCK Bottle.dmg`** (или `.app.zip`) на странице
   [**Releases**](../../releases/latest) — собирать ничего не нужно.
2. Открой `.dmg` и перетащи **PCK Bottle.app** в `/Applications`.

Сборки подписаны **ad‑hoc** (без нотаризации Apple), поэтому при первом запуске
macOS Gatekeeper может сказать, что приложение «повреждено» или «от неизвестного
разработчика». Сними карантин один раз:

```bash
xattr -dr com.apple.quarantine "/Applications/PCK Bottle.app"
```

…либо правый клик по приложению → **Открыть** → **Открыть**. (Нотаризованный
релиз открывался бы обычным двойным кликом.)

## ✨ Возможности

- Открытие игрового **`.app`** (сам находит `.pck` внутри) или отдельного **`.pck`**.
- Просмотр содержимого пакета деревом файлов и папок.
- **Drag & drop** файлов и целых папок — вложенность сохраняется, а папка‑обёртка
  вроде `translation/` распаковывается автоматически, чтобы её содержимое легло
  на нужные пути в пакете.
- Подготовленные изменения — в сворачиваемой панели **«Изменения»**
  (заменить / добавить / удалить / дублировать); на диск ничего не пишется до
  кнопки **«Запаковать»**.
- **Удаление / дублирование / копирование / вставка**, перетаскивание строк
  **в Finder** и **извлечение** выбранных файлов на диск.
- **Отмена/повтор** (⌘Z / ⇧⌘Z) для каждого изменения, с анимацией.
- Автоматическая **резервная копия** оригинала и **восстановление из копии**
  в любой момент (Файл → Восстановить из резервной копии).
- Корректная упаковка Godot: точная потоковая переупаковка, правильный паддинг
  путей и выравнивание данных по формату (Godot 3 / Godot 4), а скрытые папки
  Godot `.import` / `.godot` (импортированные текстуры) включаются в пакет.
- Локализация интерфейса — **English / Русский / 中文** — переключение из меню‑бара.
- Нативное **универсальное** приложение (Intel + Apple Silicon), macOS 10.13+.

## 🚀 Как пользоваться

1. **Открой** игру: перетащи `.app` или `.pck` в окно, либо **Файл → Открыть**.
2. **Подготовь** мод: перетащи его папку на дерево. Для русификатора перетаскивай
   саму папку **`translation/`** — её `scenarios/`, `UI/`, `.import/`, … лягут на
   соответствующие пути пакета.
3. **Проверь** панель изменений, оставь галочку **«Резервная копия оригинала»**,
   нажми **«Запаковать изменения»**.
4. Запусти игру. Откатиться — **Файл → Восстановить из резервной копии**.

## 🔧 Сборка из исходников

Нужен свежий Xcode (Swift) и Rust с обоими таргетами Apple.

```bash
rustup target add aarch64-apple-darwin x86_64-apple-darwin

# Универсальный .app (debug или release):
CONFIGURATION=release bash macos/PCKBottle/scripts/build-app.sh
# → macos/PCKBottle/build/PCK Bottle.app

# По желанию — образ диска:
bash macos/PCKBottle/scripts/make-dmg.sh

# Тесты Rust‑ядра:
cargo test --manifest-path crates/pck-core/Cargo.toml
```

Release‑сборки ремапят локальные пути и стрипают бинарники, поэтому готовый
`.app` не содержит домашнюю директорию и имя пользователя.

## 🧩 Как устроено

| Путь | Что это |
|------|---------|
| [`crates/pck-core`](crates/pck-core) | Общее **Rust**‑ядро: сканирование, чтение, извлечение и безопасная **атомарная** переупаковка PCK. Поставляется как маленький `pck-core-cli` внутри приложения. |
| [`macos/PCKBottle`](macos/PCKBottle) | Нативное macOS‑приложение на **AppKit** — поддерживаемый продукт. |
| [`legacy/`](legacy) | Устаревший UI на Tauri/Vue + JS‑тесты, оставлены для справки. Не входят в сборку. |

Приложение — тонкая оболочка AppKit, которая вызывает встроенный `pck-core-cli`,
так что вся критичная логика разбора/упаковки живёт в одном проверяемом
Rust‑крейте.
